【手順(仮説明)】
1.LiveSplitのファイルを開く (初期名だと LiveSplit_ver)

2.ComponentsフォルダにManosabaRTA_test.dllを入れる。

3.LIveSplitを起動

4.左クリック→Edit Layout → + → Control → Manosaba RTA

5.Layout Editor(Manosaba RTAを追加した画面)のTimerを右クリック → Timing Methodを「Game Time」にする
